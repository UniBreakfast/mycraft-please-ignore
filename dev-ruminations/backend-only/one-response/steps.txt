Алгоритм создания для крошечных серверов. Только бэкэнд.

1. Создаём index.js, в нём будет весь код сервера на основе модуля http.
2. Сервер на все запросы будет отвечать одним и тем же текстом. Это может быть html-разметка страницы по принципу "all inclusive", и если/когда понадобятся стили, то и CSS в тэге <style>, а если/когда понадобится поведение, то и JS в тэге <script>.

3. Опубликовать такой проект можно на любом сервере-хостинге, поддерживающем NodeJS. Например на heroku.com.
