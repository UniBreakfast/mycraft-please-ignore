Алгоритм создания для единичного файл-сервера с номинальным бэкэнд-фронтэнд взаимодействием.

1. Создаём index.js, в нём будет весь код сервера на основе модуля http.
2. Сервер на все запросы будет отвечать одним и тем же текстом - содержимым лежащего рядом файла index.html, для чего этот файл будет считываться с диска посредством модуля fs.
3. Создаём рядом index.html, в нём будет весь HTML, а если/когда понадобятся стили, то и CSS в тэге <style>, а если/когда понадобится поведение, то и JS в тэге <script>.

4. Опубликовать такой проект можно на любом сервере-хостинге, поддерживающем NodeJS. Например на render.com.
