Алгоритм создания для простого файл-сервера с самодостаточным фронтэндом, ничего кроме собственно файлов от бэкэнда не ждущим.

1. Создаём index.js, в нём будет весь код сервера на основе модуля http.
2. Сервер на все запросы будет отвечать содержимым файлов из папки public или front, для чего эти файлы будет считываться с диска посредством модуля fs. Какой именно файл запрашивается, будет определяться свойством .url из запроса. Если запрошен отсутствующий файл, будет отдаваться специальная страница 404.html
3. Создаём рядом папку public или front, в ней будут файлы фронтэнда (.html, .css, .js), организованные любым удобным образом.

4. Опубликовать такой проект можно на любом сервере-хостинге, поддерживающем NodeJS. Например на render.com.
