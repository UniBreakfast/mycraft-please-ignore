Алгоритм создания для микроскопических задач. Конечно, без бэкэнда.

1. Создаём index.html, в нём будет весь HTML, а если/когда понадобятся стили, то и CSS в тэге <style>, а если/когда понадобится поведение, то и JS в тэге <script>.
2. Другие файлы создаём лишь тогда, когда понадобятся другие странички. И то - только лишь файлы page_name.html.

3. Опубликовать можно на любом файловом сервере-хостинге. Например прямо на github.com посредством функций GitPages.
