Алгоритм создания для маленьких одностраничных сайтиков. Только фронтэнд.

1. Создаём index.html, в нём будет весь HTML главной страницы.
2. Стили задаём посредством CSS в отдельном файле style.css, подключаем его тэгом <link>.
3. Поведение задаём посредством JavaScript в отдельном файле script.js, подключаем его посредством тэга <script>.

4. Опубликовать можно на любом файловом сервере-хостинге. Например прямо на github.com посредством функций GitPages.
